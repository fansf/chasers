# 1056-lin
## 题目  
**[题目描述]**  
有一个正方形，四个角的坐标（x,y)分别是(1，-1)，(1，1)，(-1，-1)，(-1，1)，x是横轴，y是纵轴。写一个程序，判断一个给定的点是否在这个正方形内(包括正方形边界)。如果点在正方形内，则输出yes，否则输出no。   

**[输入描述]**   
输入一行，包括两个整数x、y，以一个空格分开，表示坐标(x,y)。  

**[输出描述]**  
输出一行，如果点在正方形内，则输出yes，否则输出no。  

**[输入样例]**  
1 1  

**[输出样例]**  
yes  

## 讲解  
[条件语句]([1])  

## 代码  

```cpp
#include<stdio.h>
#include<iostream>

using namespace std; 

int main()
{
	float x,y;
	cin>>x>>y;
	if(x>=-1&&x<=1&&y>=-1&&y<=1)
		cout<<"yes"<<endl;
	else
		cout<<"No"<<endl;
    return 0;
}
```
