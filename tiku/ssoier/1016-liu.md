# 1016
## 题目  
**[题目描述]**  
分别定义int，short类型的变量各一个，并依次输出它们的存储空间大小(单位：字节)。  

**[输入描述]**   
无  

**[输出描述]**  
一行，两个整数，分别是两个变量的存储空间大小，用一个空格隔开。  

**[输入样例]**  
无  

**[输出样例]**  
无  

## 讲解    
[基本数据类型](a)  

## 代码   

```cpp
#include<iostream>  

using namespace std;  

int main()  
{  
	int a;  
	short b;  
	cout<<sizeof(a)<<" "<<sizeof(b)<<endl;   
	return 0;  
}  
```
