# 1076-lin
## 题目  
**[题目描述]**  
监护室每小时测量一次病人的血压，若收缩压在90-140之间并且舒张压在60-90之间(包含端点值)则称之为正常，现给出某病人若干次测量的血压值，计算病人保持正常血压的最长小时数。  

**[输入描述]**   
第一行为一个正整数n(n<100)，其后有n行，每行2个正整数，分别为一次测量的收缩压和舒张压。  

**[输出描述]**  
输出仅一行，血压连续正常的最长小时数。  

**[输入样例]**  
4  
100 80  
90 50  
120 60  
140 90  

**[输出样例]**  
2  

## 讲解  
[数组]([1])  

[条件语句]([2])  

[迭代语句]([3])  

[运算符`&&`]([4])  

## 代码  

```cpp
#include<stdio.h>
#include<iostream>

using namespace std;

int main()
{
	int n,m=0,a[100][2],max=-1;
	cin>>n;
	for(int i=0;i<n;i++)
		for(int j=0;j<2;j++)
    		cin>>a[i][j];
	for(int i=0;i<n;i++)
	{
		if(a[i][0]>=90 && a[i][0]<=140 && a[i][1]>=60 && a[i][1]<=90)
			m++;
		else
		{
			if(m>max)
				max=m;
			m=0;
		}
	}
	cout<<max<<endl;
	return 0;
}
```
