# 1059-lin
## 题目  
**[题目描述]**  
班上有学生若干名，给出每名学生的年龄（整数），求班上所有学生的平均年龄，保留到小数点后两位。  

**[输入描述]**   
第一行有一个整数n（1≤ n ≤ 100），表示学生的人数。其后n行每行有1个整数，表示每个学生的年龄，取值为15到25。  

**[输出描述]**  
输出一行，该行包含一个浮点数，为要求的平均年龄，保留到小数点后两位。  

**[输入样例]**  
2  
18  
17  

**[输出样例]**  
17.5

## 代码  

```cpp
#include<stdio.h>
#include<iostream>

using namespace std;

int main()
{
	int n,c;
	float a,sum=0;
	cin>>n;
	c=n
	while(c)
	{
		cin>>a;
		sum+=a;
		c--;
	}
	printf("%.2f",sum/n);
	return 0;
}
```
